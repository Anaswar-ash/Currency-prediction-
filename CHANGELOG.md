# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-11-05

### Changed
- Improved the frontend user interface by integrating Bootstrap.
- Redesigned the layout for a more modern and professional look.
- Made the prediction chart responsive.
- Updated the `README.md` with the latest changes.

## [0.1.1] - 2025-11-02

### Changed
- Updated `README.md` with detailed setup instructions for Termux users, including the use of `proot-distro` for the backend to resolve `tensorflow` installation issues.

## [0.1.0] - 2025-11-02

### Added
- Initial project setup.
- Basic backend structure with Flask.
- LSTM model for currency prediction.
- `.gitignore` file.
